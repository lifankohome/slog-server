# slog-server

Receive and publish real time log.

### Start server

 + npm install
 + npm start

The console will output:

```text
> node server.js

[INFO]@[ws] Starting...

[INFO]@[ws] Started

[INFO]@[http] Starting...

[INFO]@[http] Started
```
